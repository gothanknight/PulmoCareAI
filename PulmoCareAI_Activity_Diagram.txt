DECISION POINTS:
================

{Authentication Valid?}
├── YES → Continue to Main Application
└── NO → Show Error and Return to Login

{File Format Valid?}
├── YES → Proceed with Analysis
└── NO → Show Error and Request New File

{Analysis Complete?}
├── YES → Display Results and Save
└── NO → Show Error and Retry

{User Wants to Export?}
├── YES → Generate PDF/CSV Report
└── NO → Continue with Current View

{User Wants to Logout?}
├── YES → Clear Session and Redirect
└── NO → Continue with Current Session
